# 📘 Exercícios APC | Exercises for APC discipline

>  Repositório acadêmico com exercícios de Algoritmos e Programação de Computadores focado no desenvolvimento de lógica e fundamentos em C e Python.

>  Academic repository containing Computer Algorithms and Programming exercises focused on logic development and core foundations using C and Python.

---

##  Sobre

Este projeto reúne listas de exercícios resolvidos durante a disciplina de Algoritmos e Programação de Computadores (APC).  
O foco está na consolidação de fundamentos essenciais de programação, incluindo:

- Lógica de programação  
- Estruturas condicionais e de repetição  
- Vetores e matrizes  
- Funções e modularização  
- Organização e boas práticas de código  

Os exercícios estão organizados por listas e podem ser compilados e executados individualmente.

---

## About

This project contains solved exercise lists developed during the Computer Algorithms and Programming (APC) course.  
The focus is on strengthening core programming foundations, including:

- Programming logic  
- Conditional and loop structures  
- Arrays and matrices  
- Functions and modularization  
- Code organization and basic best practices  

Exercises are organized by lists and can be compiled and executed individually.
